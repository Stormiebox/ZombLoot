# ZombLoot

**ZombLoot** is a dynamic, event-driven loot dropping mod for Project Zomboid. It introduces rewarding new mechanics that give players exciting incentives to hunt down zombies through customizable drop chances, timed events, extreme weather, and kill streaks.

Developed by **Stormbox**.

## Features

ZombLoot is divided into four modular mechanics, all fully configurable via the game's Sandbox Options:

### 🧟 Core ZombLoot
A baseline dynamic loot system. Set a global chance for zombies to drop specified items upon death. You can configure whether the item drops directly onto the zombie's corpse or goes straight into the killer's inventory.

### ⏰ ZombTime
Designate a specific in-game time window as "Loot Bonus Time." During these hours, killing zombies grants a chance to receive special loot drops. Server-wide notifications alert players when Fever Time begins and ends!

### ⛈️ ZombWeather (Extreme Weather Drops)
Brave the elements for unique rewards! This feature provides a special loot table incentive for players who hunt zombies during heavy rain, snow, or thick fog. 

### ⚔️ ZombCombo (Kill Streaks)
Rewards aggressive and fast-paced gameplay. If a player achieves a consecutive number of rapid kills within a specific time window, they are guaranteed a combo bonus drop once the combo target is reached. 

## Configuration & Sandbox Options

Server admins and single-player users can extensively configure the mod under the **Sandbox Options** menu. 

**Key Settings Include:**
*   **Drop Chances:** Fine-tune the exact drop probabilities for each event type.
*   **Drop Items:** Specify the exact items to be dropped using their Item IDs. Separate multiple items with a slash `/` (e.g., `Base.Cookie/Base.Muffin/Base.Axe`). The mod will randomly pick one from the list.
*   **Drop Location:** Choose whether the reward spawns on the **Zombie Corpse** or directly into the player's **Belongings**.
*   **Text & Notifications:** Enable or disable text notifications ("Halo Notes") and customize the text displayed for drops, combo bonuses, and Fever Time announcements.

## Multiplayer Compatibility

This mod is fully compatible with Multiplayer and Dedicated Servers. Halo notes and timed events are broadcasted safely across servers to ensure all active players are properly notified of events like Fever Time.

## Installation

1. Subscribe to the mod on the Steam Workshop (or extract the mod folder into your `Zomboid/mods` directory).
2. Enable the mod in your save file or server settings.
3. Configure your desired loot rules via the Sandbox Options.
